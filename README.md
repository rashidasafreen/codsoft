# Codsoft — Machine Learning Internship Projects

Machine learning projects completed during the Codsoft internship program. Includes fraud detection, Iris flower classification, and sales prediction models built with Python and scikit-learn.

## Projects

### 1. Fraud Detection (`Fraud__Detection.ipynb`)
Binary classification model to detect fraudulent transactions.
- **Algorithm:** Random Forest / Logistic Regression
- **Key steps:** Data preprocessing, feature engineering, handling class imbalance (SMOTE), model evaluation with precision/recall/F1

### 2. Iris Flower Classification (`iris_flower.ipynb`)
Classic multi-class classification on the Iris dataset.
- **Algorithm:** KNN, Decision Tree, SVM
- **Key steps:** EDA, feature visualization, model comparison, accuracy evaluation

### 3. Sales Prediction (`sales_prediction.ipynb`)
Regression model to predict product sales from advertising spend.
- **Algorithm:** Linear Regression
- **Key steps:** Correlation analysis, feature selection, model training, RMSE evaluation

## Tech Stack

Python, NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook
